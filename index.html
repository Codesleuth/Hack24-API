Teams Resource

Allows creating and modifying teams and team members

/teams get post

get
post

get /teams

  • Response

HTTP status code 200

Body

Type: application/hal+json

Example:

{
  "_links": {
    "self": {
      "href": "/teams"
    }
  },
  "_embedded": {
    "item": [
      {
        "_links": {
          "self": {
            "href": "/teams/refactor-tractor"
          },
          "members": {
            "href": "/teams/refactor-tractor/members"
          }
        },
        "teamid": "refactor-tractor",
        "name": "Refactor Tractor"
      },
      {
        "_links": {
          "self": {
            "href": "/teams/code-monkeys"
          },
          "members": {
            "href": "/teams/code-monkeys/members"
          }
        },
        "teamid": "code-monkeys",
        "name": "Code Monkeys"
      }
    ]
  }
}

/teams/{teamId} get

Returns a specific team.

get

get /teams/{teamId}

  • Request
  • Response

URI Parameters

  • Team ID: required (string)

HTTP status code 200

Body

Type: application/hal+json

Example:

{
  "_links": {
    "self": {
      "href": "/teams/refactor-tractor"
    },
    "members": {
      "href": "/teams/refactor-tractor/members"
    }
  },
  "_embedded": {
    "members": {
      "_links": {
        "self": {
          "href": "/teams/refactor-tractor/members"
        }
      },
      "item": [
        {
          "_links": {
            "self": {
              "href": "/users/barry"
            }
          },
          "userid": "barry",
          "name": "Barry",
          "joined": "2016-01-23T12:00:00.0000000Z"
        },
        {
          "_links": {
            "self": {
              "href": "/users/mary"
            }
          },
          "userid": "mary",
          "name": "Mary",
          "joined": "2016-01-21T12:00:00.0000000Z"
        }
      ]
    },
    "teamid": "refactor-tractor",
    "name": "Refactor Tractor"
  }
}
